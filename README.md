# 🌐 Django Project

## 📌 Project Overview

This is a web application built using the **Django framework**.
The project demonstrates backend development, database handling, and dynamic web features.

---

## 🚀 Features

* User-friendly interface
* Django MVC (MVT) architecture
* Database integration (SQLite)
* Authentication system (Login/Signup)
* CRUD operations (Create, Read, Update, Delete)

---

## 🛠️ Tech Stack

* **Backend:** Python, Django
* **Frontend:** HTML, CSS, JavaScript
* **Database:** SQLite

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```
git clone https://github.com/40-taniya/StudentTaskManager.git
cd StudentTaskManager
```

### 2️⃣ Create Virtual Environment 

```
python -m venv env
source env/bin/activate   # Linux/Mac
env\Scripts\activate      # Windows
```

### 3️⃣ Install Dependencies

```
pip install -r requirements.txt
```

### 4️⃣ Apply Migrations

```
python manage.py migrate
```

### 5️⃣ Run Server

```
python manage.py runserver
```

👉 Open browser and go to:
(http://127.0.0.1:8000/)
